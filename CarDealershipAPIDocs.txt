Car Dealership API Documentation
This API provides endpoints to manage car dealerships, cars, and deals. It uses the Express framework and Faker.js library to generate sample data.

Base URL: http://localhost:3000
Endpoints:
1. View All Cars
Endpoint: GET /api/cars
Description: Retrieves a list of all available cars.
Request: None
Response Format:
json
Copy code
[
  {
    "id": "string",
    "name": "string",
    "dealership": "string",
    "owner": "string | null"
  },
  ...
]
2. View All Cars in a Dealership
Endpoint: GET /api/cars/dealerships/:dealership
Description: Retrieves a list of cars available in a specific dealership.
Request: None
Response Format:
json
Copy code
[
  {
    "id": "string",
    "name": "string",
    "dealership": "string",
    "owner": "string | null"
  },
  ...
]
3. View All Deals from a Certain Dealership
Endpoint: GET /api/dealerships/:dealership/deals
Description: Retrieves a list of deals provided by a certain dealership.
Request: None
Response Format:
json
Copy code
[
  {
    "id": "string",
    "carId": "string",
    "dealership": "string",
    "price": "number"
  },
  ...
]
4. Buy a Car After a Deal Is Made
Endpoint: POST /api/buy/:dealId
Description: Allows the user to buy a car after a deal is made.
Request Format:
json
Copy code
{
  "dealId": "string"
}
Response Format:
json
Copy code
{
  "message": "string",
  "car": {
    "id": "string",
    "name": "string",
    "dealership": "string",
    "owner": "string"
  }
}
